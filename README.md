# Superstore Sales & Customer Analytics Dashboard

## Project Overview

Developed a two-page interactive Power BI dashboard using the Superstore dataset to analyse sales performance, profitability, customer behaviour, and operational metrics.

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
* Excel

## Key KPIs

* Total Sales
* Total Profit
* Total Orders
* Profit Margin %
* Average Order Value

## Dashboard Features

### Executive Sales Dashboard

* Sales Trend Analysis (2014–2017)
* Profit Trend Analysis (2014–2017)
* Regional Sales Performance
* Top 10 Products by Sales
* Sales Distribution by Category
* Interactive Region Filtering

### Customer & Operations Analytics Dashboard

* Top 10 Customers by Sales
* Sales Distribution by Segment
* Top Sub-Categories by Profit
* Sales by Ship Mode

## Skills Demonstrated

* Data Cleaning
* Data Transformation
* Data Modeling
* DAX Measures
* KPI Development
* Dashboard Design
* Data Visualization
* Business Intelligence

## Dataset

Superstore Sales Dataset

## Project Objective

To transform raw sales data into meaningful business insights through interactive dashboards, enabling better understanding of sales performance, customer behaviour, product profitability, and operational efficiency.
